# feishu-notice-bot
* 使用飞书开放平台自定义ChatOps机器人，功能主要为同步github动态的消息推送
* 自定义机器人使用 webhook 从外部系统接收消息，然后将消息推送到群聊。自定义机器人用于自动发送通知，不会回复消息。自定义机器人只能在群聊中使用。
* Deploy on AWS server
